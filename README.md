# neural-networking-in-python
Advanced Self-Driving Car SimulatorA high-performance 2D self-driving car simulation built with Python and Pygame. The project utilizes a Genetic Algorithm to evolve neural network "brains" that learn to navigate a custom racetrack through trial and error.
# 🚀 FeaturesNeural Network Integration: Each car is powered by a multi-layer perceptron (MLP) with 9 sensor inputs and 3 control outputs (steering, throttle, brake).Genetic Algorithm: Implements tournament selection, elitism, and crossover/mutation strategies to evolve the population.Realistic UI: Features a custom-built HUD with live performance graphs, real-time speedometers, and simulation controls.Dynamic Simulation: Adjustable simulation speeds, sensor visualization, and a "Replay Mode" to watch the best-evolved agent.Ray-Casting Sensors: Realistic obstacle detection using 9 individual ray-casts to calculate proximity to walls.

# 🧠 Technical Architecture
# 🧬 The Genetic AlgorithmThe trainer manages a population of 60 cars per generation. Fitness is calculated based on:Number of checkpoints passed.Total distance traveled.Average speed (to discourage overly cautious driving).Time alive.🕸️ Neural Network StructureThe simulation uses a feed-forward neural network with the following architecture:Input Layer: 13 neurons (9 sensors + speed, angle to checkpoint, distance, and time).Hidden Layers: Two hidden layers (16 and 12 neurons) using tanh activation.Output Layer: 3 neurons (Steer, Throttle, Brake).📋 PrerequisitesPython 3.xNumPyPygameInstall dependencies via pip:Bashpip install pygame numpy
